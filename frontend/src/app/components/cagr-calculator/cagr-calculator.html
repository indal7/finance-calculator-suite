How CAGR Calculation Works

CAGR is calculated using:

CAGR = (Ending Value / Beginning Value)^(1/n) − 1

Where n = number of years.

What is CAGR?

CAGR (Compound Annual Growth Rate) represents the rate at which an investment would have grown if it grew at a steady rate annually. Unlike absolute returns, CAGR accounts for the effect of compounding and the time period — making it ideal for comparing mutual funds, stocks, and portfolios.

CAGR vs Absolute Return

Absolute return shows total percentage gain regardless of time. CAGR normalises it per year. A 100% absolute return over 2 years is a 41.4% CAGR — very different from 100% over 10 years (7.2% CAGR).

Example Calculation

Investment grew from ₹50,000 to ₹1,00,000 in 5 years → CAGR = 14.87%, Absolute Return = 100%.

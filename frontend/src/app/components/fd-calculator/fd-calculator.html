How FD Calculation Works

FD maturity amount is calculated using compound interest:

A = P × (1 + r/n)^(n×t)

Where P = principal, r = annual rate, n = compounding frequency, t = years.

Example

₹1,00,000 at 7% for 3 years (quarterly) → Maturity: ₹1,23,145, Interest earned: ₹23,145.
